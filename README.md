# This is my personal page

## Installation

Global installation:

```
sudo apt update
sudo apt install -y python3-pip \
		python3-dev \
		python3-venv \
		libpq-dev \
		postgresql \
		postgresql-contrib \
		nodejs \
		npm
```
Requirements:

`pip install -r requirements.txt`

## Run webpack
`npm install`

`npx webpack --config webpack_config.js`
